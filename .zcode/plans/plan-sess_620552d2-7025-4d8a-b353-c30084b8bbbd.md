### Step 2 — Scaffold-from-component button [#5]

**New message type** in `src/types.ts`:
```ts
export type ScaffoldPropMappingsHandler = {
  name: 'SCAFFOLD_PROP_MAPPINGS';
  handler: (result: { ok: boolean; mappings?: ...; message?: string }) => void;
};
```

**`src/main.ts`:** register a handler that, for the current selection, reads `componentPropertyDefinitions` from the main component/component set, and for each `VARIANT` property builds `{ [option]: { prop: propertyName, value: option } }` for every entry in `variantOptions`. Returns the generated mapping object (or an error message if nothing scaffoldable / no selection).

**`src/ui.tsx`:** Add a secondary **"Generate from component"** button next to the "Prop mappings JSON" field label (or directly above the textarea). On click, emit `SCAFFOLD_PROP_MAPPINGS`, then **merge** the returned mappings into the existing parsed JSON (existing keys preserved, new keys added), and re-serialize into the textarea. If the textarea has invalid JSON, surface the error via `errorMessage` and don't overwrite.
